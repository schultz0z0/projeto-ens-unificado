# Estratégias de Tráfego Pago para Setor de Educação (Seguros)

## Estrutura de Funil "Ampulheta"

### 1. Topo de Funil (Consciência) - "O Despertar"
- **Objetivo:** Tráfego frio para conteúdo de alto valor (Blog/Vídeo).
- **Criativos:** Cortes de podcast sobre tendências, notícias de sinistros grandes, curiosidades sobre apólices.
- **Métrica:** CTR (Taxa de Clique) e Video View 50%.
- **Dica Técnica:** Use o Pixel para criar público de "Visualizou 50% do vídeo".

### 2. Meio de Funil (Consideração) - "A Troca"
- **Objetivo:** Captura de Lead (CPL).
- **Oferta (Lead Magnet):**
  - E-book: "Guia de Riscos Cibernéticos para Corretores".
  - Checklist: "Como auditar uma apólice empresarial".
  - Masterclass Gravada: Aula magna gratuita de um curso pago.
- **Tracking:** Evento `Lead` no servidor (CAPI) é crucial aqui para otimizar o algoritmo.

### 3. Fundo de Funil (Conversão) - "A Oferta"
- **Objetivo:** Venda / Matrícula (ROAS).
- **Público:** Remarketing de quem baixou o E-book ou visitou a página do curso (Checkout Abandonado).
- **Copy (PAS):** "Você baixou o guia, mas ainda não domina a técnica..."
- **Oferta:** Desconto condicional, Bônus exclusivo, Garantia de satisfação.
