# ENS RAG Marketing

Use this skill for:

- campaign ideas
- copy patterns
- audience notes
- validated creative directions
- approved marketing learnings

## Retrieval

- Search `marketing` first.
- Bring in `courses` when the work is for a specific course.
- Bring in `insights` when campaign or funnel evidence matters.

## Save Rules

Do not save drafts automatically.

Use `ens_rag_save_marketing_memory` only when:

- the user explicitly validated the result
- the validation can be summarized in a short note
- the saved knowledge is reusable

If the user has not validated the output yet, keep it out of the RAG.
